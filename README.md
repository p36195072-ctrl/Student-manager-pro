Student Manager Pro (JSON Edition)

A beginner-to-intermediate Python project that demonstrates how to build a complete Student Management System with permanent JSON storage.

📌 Description

This project allows users to:

Add students
Search students
View all students
Delete students
Edit student details
Save data automatically
Load data automatically
Exit safely

Student records are stored in students.json, so data remains available even after the program closes.
